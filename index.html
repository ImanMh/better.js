Better.js for a Better Javascript in Javascript

Javascript is everywhere but it may become hard to debug sometime. better.js library provide the tools to help you debug your javascript and keep it running smoothly.

Go Github

All in javascript

As it is all in javascript, no other dependancy that you already got. Easy to run where you need it

Provide Missing Features

Strong type checking, variable range monitoring, garbage collection avoidance, better.js got many feature to help you debug your code and be sure it does what you expect.

Browser and server

As it is pure javascript, you can run it in server and in browser.
